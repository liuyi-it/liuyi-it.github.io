MemPalace

本地优先的 AI 记忆系统

  • Python
  • ChromaDB
  • MCP
  • RAG
  • MIT

MemPalace 将项目资料与 AI 对话整理成可检索的“记忆宫殿”，帮助 AI 在后续会话中找回决策、背景、调试记录和架构讨论。

项目强调本地运行、无需外部 API Key、保留原始上下文，并通过语义搜索让长期信息重新可用。

定位：AI memory / semantic search / developer tooling 运行方式：CLI + MCP-compatible tools
